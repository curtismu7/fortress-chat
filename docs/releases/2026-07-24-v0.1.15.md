## Signing / Notarization

Set these vars in CI/release shell to enable notarization in electron-builder `afterSign` hook:

- `APPLE_ID`
- `APPLE_APP_SPECIFIC_PASSWORD`
- `APPLE_TEAM_ID`

If missing, notarization is skipped and local dev builds still succeed.
